# src

Esta pasta contem o codigo-fonte principal do CircuitStock.

## O que colocar aqui

- Rotas, paginas e layouts da aplicacao.
- Componentes React usados pela interface.
- Codigo de dominio e integracoes futuras, quando forem criados.

## O que evitar

- Arquivos gerados automaticamente.
- Dados locais de estoque ou banco SQLite.
- Documentacao extensa que pertença a `docs/`.

## Estado atual

Atualmente o codigo esta concentrado em `src/app`, usando o App Router do Next.js. Ainda nao ha banco de dados, Prisma, CRUD ou regras de negocio implementadas.
