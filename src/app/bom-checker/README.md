# src/app/bom-checker

Rota prevista para comparar uma lista de materiais com o estoque disponivel.

## O que colocar aqui

- Telas para importar, revisar ou colar itens de uma BOM.
- Componentes de comparacao entre necessidade do projeto e disponibilidade.
- Estados de itens suficientes, faltantes ou parcialmente disponiveis.

## O que evitar

- Criar parser definitivo antes de definir o formato de entrada suportado.
- Misturar cadastro de componentes com a analise da BOM.
- Versionar listas reais de materiais ou dados sensiveis de projetos.

## Estado atual

A rota contem apenas uma pagina placeholder. O BOM Checker sera implementado em etapas futuras, depois que o estoque local estiver modelado.
