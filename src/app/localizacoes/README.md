# src/app/localizacoes

Rota prevista para registrar os locais fisicos onde os componentes ficam guardados.

## O que colocar aqui

- Telas de armarios, gavetas, caixas, prateleiras ou posicoes fisicas.
- Componentes especificos para visualizar e manter localizacoes.
- Estados de consulta e organizacao fisica do laboratorio.

## O que evitar

- Misturar movimentacoes de estoque nesta rota.
- Guardar mapas, fotos ou anexos pesados sem criterio definido.
- Registrar dados locais sensiveis no Git.

## Estado atual

A rota contem apenas uma pagina placeholder. O cadastro de localizacoes sera implementado em etapas futuras.
