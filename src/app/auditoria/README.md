# src/app/auditoria

Rota prevista para conferencia fisica e ajuste controlado de quantidades.

## O que colocar aqui

- Telas de abertura e acompanhamento de auditorias.
- Componentes para comparar quantidade registrada e quantidade conferida.
- Estados de pendencia, divergencia e conclusao, quando forem definidos.

## O que evitar

- Ajustes silenciosos de estoque sem registro.
- Misturar cadastro comum de componentes com fluxo de auditoria.
- Dados reais de conferencias no repositorio.

## Estado atual

A rota contem apenas uma pagina placeholder. O fluxo de auditoria sera desenhado apos a modelagem inicial do estoque.
