# Changelog

O CircuitStock usa versionamento por series de evolucao. Cada versao representa um pacote coerente de maturidade do produto, e nao apenas uma funcionalidade isolada.

## [0.5.0] - Fundacao com cadastro inicial de componentes

Marco da serie 0.x que consolida a fundacao do produto com estrutura tecnica, documentacao oficial e primeiros fluxos reais de inventario.

### Base tecnica

- Base Next.js com App Router.
- TypeScript.
- Tailwind CSS.
- Prisma.
- SQLite local.
- Repositorio GitHub publico.

### Produto e interface

- Layout base.
- Navegacao lateral.
- Documentacao estrutural.
- Visao do produto.
- Roadmap por series.

### Dados e fluxos

- Seed de categorias e localizacoes.
- Dashboard com dados reais.
- Categorias com dados reais.
- Localizacoes com dados reais.
- Componentes com listagem real.
- Cadastro inicial de componentes.
- Validacoes basicas.
- Movimentacao ENTRY no cadastro inicial.
