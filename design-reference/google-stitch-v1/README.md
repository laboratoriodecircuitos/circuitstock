# Google Stitch v1 - Referencia Visual

Esta pasta contem a referencia visual oficial v1 do CircuitStock criada no Google Stitch.

A referencia ainda nao foi aplicada ao front atual. Ela existe para preservar a direcao visual aprovada e orientar uma etapa futura de refinamento da interface.

## Arquivos

- `dash_code.html`: exportacao HTML/CSS da tela criada no Google Stitch.
- `DASH_DESIGN.md`: especificacao de design exportada ou copiada da referencia.
- `dash_screen.png`: captura visual da tela aprovada.
- `DESIGN_NOTES.md`: sintese tecnica local para orientar a adaptacao futura.

## O que preservar

A implementacao futura deve usar esta referencia para preservar, com a maior fidelidade possivel:

- cores;
- fontes;
- espacamentos;
- composicao geral;
- hierarquia visual;
- estilo de cards;
- estilo de navegacao;
- aparencia profissional de ferramenta de laboratorio.

## Como usar no futuro

A referencia deve ser adaptada para a stack atual do CircuitStock: Next.js, TypeScript e Tailwind CSS.

O codigo exportado pelo Google Stitch nao deve ser copiado de forma cega. Ele pode conter convencoes, dependencias externas, classes ou estruturas que nao se encaixam diretamente no projeto. A etapa futura deve reinterpretar o design dentro dos padroes existentes do repositorio, respeitando acessibilidade, responsividade, dados reais e componentes ja implementados.

O objetivo e replicar a experiencia visual com maxima fidelidade possivel sem comprometer a estabilidade tecnica do produto.
